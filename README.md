mywm - my window manager
---------------------------------------------------------------------------------
mywm is a small reparenting window manager

it's meant to be easily modified

requirements
---------------------------------------------------------------------------------
xcb header files: icccm, shape, ewmh

X11 header files: keysym

installation
---------------------------------------------------------------------------------
install using 

	make install clean
